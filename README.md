Projet Vite

Ce projet a été réalisé en utilisant Vite, un outil moderne et rapide pour le développement front-end.

Technologies utilisées

- Vite

- HTML

- JavaScript

- CSS

Structure du projet

├── index.html        # Fichier principal à la racine du projet
└── src
    ├── main.js       # Code JavaScript principal
    ├── style.css     # Styles CSS du projet
    └── font          # Dossier contenant les polices (fonts)


Le fichier index.html se trouve à la racine du projet.

Le code JavaScript principal est dans src/main.js.

Les styles CSS sont dans src/style.css.

Les polices utilisées dans le projet sont stockées dans src/font.

Lancer le site : 

Le site est hébergé via GitHub Pages.

Étapes pour accéder au site publié :

1. Aller sur le dépôt GitHub du projet

2. Cliquer sur Settings (Paramètres)

3. Aller dans la section Pages

4. Un lien vers le site publié est affiché

5. Cliquer sur ce lien pour accéder au site en ligne
